## Things I'm building

- **[squadai](https://github.com/PedroMosquera/squadai)**: one config, every
  AI coding agent, zero drift. Declare your project's operating model once and
  `squadai apply` writes the native files for Claude Code, Cursor, OpenCode,
  Windsurf, Copilot, and friends.
- **[brainbridge](https://github.com/PedroMosquera/brainbridge)**: an
  interactive wizard that sets up an Obsidian vault or plain markdown folder
  as a shared knowledge base for your AI tools.
- **[mac-telemetry-audit](https://github.com/PedroMosquera/mac-telemetry-audit)**:
  a read-only script that shows you what's watching, monitoring, or phoning
  home from your Mac.
- **[fragwise](https://github.com/PedroMosquera/fragwise)**: open-source
  fragrance discovery with an AI guide. Early days.
